## vue-tiny-cli

完成一个vue-cli的简单脚手架，快速生成应用模板，安装项目，启动服务，以及可以自动配置路由路径。

### 项目命令

- **licop init < name >:**  初始化项目
- **licop refresh:**  更新项目路由
